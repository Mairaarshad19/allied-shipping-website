# Allied Shipping Agency Website

## Project Overview

This is a modern corporate website for Allied Shipping Agency, a freight forwarding and cargo booking agency based in Lahore, Pakistan.

The purpose of this website is to build trust, showcase services, and encourage potential customers to contact the company via WhatsApp or phone.

This is NOT an e-commerce website.

This is NOT a logistics management system.

This is NOT a shipment tracking portal.

The primary goal is lead generation.
